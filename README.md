## nest install

```
// nestjs 설치
$ npm i -g @nestjs/cli
// 프로젝트 생성
$ nest new project-name
// 컨트롤러 생성
$ nest g controller cats
```

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
